# TSC Fullstack

## Development

- Install backend deps at the repo root:
  ```bash
  npm install
  ```
- Install frontend deps:
  ```bash
  npm --prefix Frontend install
  ```

### Run locally

- Full stack (frontend + backend):
  ```bash
  npm run dev
  ```
- Frontend only (Next.js):
  ```bash
  npm run dev:frontend
  ```
- Backend only:
  ```bash
  npm run dev:backend
  ```

## Build

```bash
npm run build
```

## Start (production)

- Frontend:
  ```bash
  npm run start:frontend
  ```
- Backend:
  ```bash
  npm run start:backend
  ```
